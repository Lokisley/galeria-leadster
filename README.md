# LeadPexels

Esse projeto foi criado como atividade técnica para processo seletivo de Desenvolvedor Front-End na Leadster.

Para o desenvolvimento foi utilizado o framework React, juntamente com boostrap para responsividade e SASS como pré-processador CSS.

O projeto consiste em uma galeria de imagens, que foram obtidas a partir do consumo da API Pexels.

## Pexels API

Para execução do projeto, é necessário obter uma chave da API para realizar as requisições. Esta chave pode ser obtida clicando [aqui](https://www.pexels.com/pt-br/api/) e preenchendo o formulário.

Após obter sua chave da API, basta editar o objeto presente em `src\key-example.json` e renomear o arquivo para `key.json` para que o código possa ser propriamente lido e utilizado pelo aplicativo.
